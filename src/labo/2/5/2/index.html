Voorbeeld van Definition List

TCP
Transmission Control Protocol is een methode (protocol) die gebruikt wordt samen met het Internet Protocol (IP) om gegevens te zenden en delen, packets genoemd, tussen computers op het internet.
IP
Internet Protocol is een aantal regels die controleren hoe gegevens tussen computers op het internet verzonden worden. Elke computer wordt uniek bepaald met een IP adres.
FTP
File Transfer Protocol is een protocol gemaakt om bestanden uit te wisselen tussen computers op het internet.
HTTP
Hypertext Transfer Protocol is het protocol gebruikt voor uitwisseling van tekst, figuren, geluid, video en andere multimedia-bestanden op het Web.
